# speech-recognition
The following code enables you to connect to your microphone and convert your speech into text. It is a simple project that can greatly benefit those with a disability.

before downloading project - make sure to go to your terminal and enter:
pip install speedrecognition (or py -m pip install speechrecognition)
and
pip install pyttsx3 (or py -m pip install pyttsx3) to download the libraries.
